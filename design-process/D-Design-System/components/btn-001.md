# Component: **btn-001 (Titan-Primary)** 🚀

## 1. Description
The primary call-to-action button for the **"Hybrid-Engine"** experience. Designed to command attention with a high-glow, premium "Titan-grade" feel.

## 2. Visual Blueprint
- **Color**: `bg-blue-600` (Hover: `bg-blue-500`)
- **Radius**: `24px` (Small/Medium), `32px` (Large)
- **Shadow**: `shadow-[0_0_40px_rgba(59,130,246,0.3)]` (Glow effect)
- **Typography**: Inter / Bold / Italic / Uppercase

---

<button class="group relative px-10 py-5 bg-blue-600 hover:bg-blue-500 text-white rounded-[24px] font-black text-lg transition-all transform hover:scale-105 shadow-[0_0_40px_rgba(59,130,246,0.2)] tracking-tighter italic uppercase">
  <span class="relative z-10">Experience the 10s Flow</span>

## 4. Variants & States

### 4.1 Regular / Active
- **Description**: Standard CTA.
- **Visual**: Solid Blue + Under-glow.

### 4.2 Hover State
- **Description**: Triggered on cursor entry.
- **Visual**: Scale 105% + Expanded Blue Glow.

### 4.3 Pressed / Active
- **Description**: On mouse down.
- **Visual**: Scale 98% + Shadow Intensity reduction.
